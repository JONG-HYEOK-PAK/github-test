# MPPT 알고리즘 비교 시뮬레이션 초기 조건

P&O(Perturb and Observe) 및 INC(Incremental Conductance) 알고리즘의 성능(추종 속도, 정상 상태 진동 등)을 공정하게 비교하기 위한 베이스라인(Baseline) 설정값입니다.

---

## 1. 알고리즘 제어 변수 (Algorithm Parameters)
두 알고리즘 객체 생성 시 동일하게 적용되는 제어 설정입니다.

* **초기 전압 (V_init):** 15.0V
  * **설명:** 추종을 시작하는 초기 지점입니다. 0V부터 시작할 수도 있으나, 현실적인 구동 환경을 반영하여 패널 개방 전압의 절반 정도 지점에서 시작합니다. 동일한 출발선에서 시작해야 목표 도달 시간을 정확히 비교할 수 있습니다.
* **스텝 크기 (Step Size / ΔV):** 0.5V
  * **설명:** 1회 알고리즘 주기당 미세 조정하는 전압의 크기입니다. 정상 상태(Steady-state) 도달 후 두 알고리즘이 각각 얼마나 안정적인지(진동 폭)를 비교하는 핵심 통제 변수입니다.
스텝 간격
1번 조건 0.01s
2번 조건 0.05s 
---

## 2. 환경 조건 (Environmental Conditions)
알고리즘이 동작하는 외부의 가상 환경 설정입니다. 시뮬레이션 중 이 값들을 고정하여 일관된 테스트 환경을 유지합니다.

* **일사량 (Irradiance):** 1000 W/m²
  * **설명:** 구름 없는 맑은 날 정오 수준의 표준 테스트 조건(STC) 일사량입니다.
* **온도 (Temperature):** 25°C
  * **설명:** 태양광 패널의 표면 온도입니다. 온도는 패널의 출력 전압에 직접적인 영향을 미치므로 상수로 고정합니다.

---

## 3. 대상 패널 스펙 (PV Module Specs)
시뮬레이션에 사용될 가상의 250W급 태양광 패널 모델 기준값입니다. 알고리즘이 최종적으로 찾아내야 할 물리적인 정답지에 해당합니다.

* **최대 전력 (P_max):** 250W
* **최대 전력점 전압 (V_mp):** 30.0V 
  * **참고:** 알고리즘이 최종적으로 수렴해야 하는 목표 전압입니다.
* **개방 전압 (V_oc):** 37.0V 
  * **참고:** 전류가 0A일 때 걸리는 시스템의 최대 전압 한계치입니다.
* **단락 전류 (I_sc):** 8.5A 
  * **참고:** 전압이 0V일 때 흐를 수 있는 최대 전류치입니다.

일사량 변동
0초 ~ 3초 (안정기): 일사량 1000 W/m² 유지 (초기 목표점 도달 및 진동 확인)

3초 ~ 6초 (구름 통과): 일사량 500 W/m² 로 급락 (갑자기 줄어든 일사량에 대한 추종 속도 확인)

6초 ~ 10초 (다시 맑아짐): 일사량 1000 W/m² 로 급등 (P&O가 방향을 잃고 헤매는지, INC가 안정적으로 쫓아가는지 확인)
